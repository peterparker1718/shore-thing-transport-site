# Shore Thing Transport

Accessible static website for Shore Thing Transport.

## Description
This repository contains the Shore Thing Transport booking-request website for airport transfers, private rides, local transportation, and event transportation.

## Hosting
This site is being launched on Vertex hosting first, then the custom domain will be connected later.

## Current features
- Responsive landing page
- Booking request form
- Vehicle options
- Service areas
- Contact information
- Accessibility-friendly structure

## Planned upgrades
- Google Places autocomplete
- Quote logic
- Email routing
- Payments
- Custom domain connection

## Brand notice
The Shore Thing Transport name, branding, business identity, contact details, and custom written content may not be reused without permission.
